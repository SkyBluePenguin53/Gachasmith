Gacha Program (Gachasmith)
Working program of Gachasmith: a text-based gacha game written in Java.

Gachasmith is inspired by gacha games and is designed to be a free gacha game.

You are a blacksmith running a weapons business. Your goal is to earn 1000000 Mora per day.
The prices are set by the laws of your kingdom and you can't change it, so do everything you can to earn.

Design patterns used:
1. Factory (located in weaponType and materialType classes)

Coding practices observed:
1. Java naming conventions used.
2. Comments placed in blocks of code.
3. Simple but obvious names for classes and methods.
4. Minimized redundant codes.
